# ☕ coffee-script & react-js

## about

Base template for react-js on coffee-script

core: [__esbuild__](https://esbuild.github.io/)


## support

* routing: ✅

* hot reload: ✅

* sass & stylus: ✅

## run

for development `npm run dev`

for build `npm run build`

for production `npm run start`
